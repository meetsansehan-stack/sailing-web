## 배경 (왜 이 논의가 시작됐나)

이미지 폴백 작업(A) 중 출처 얘기로 분기. 핵심 재구성 2가지가 깔림:

1. **텍스트 vs 이미지는 저작권 문제가 완전히 다름.**
   - 텍스트 = 우리 모델이 "요약·재서술 + 링크"(전문 복제 X) → 출처 누구든 **거의 안전**. "저작권 때문에 텍스트 못 가져온다"는 병목 아님.
   - 이미지 = 통째 복제(요약 불가) → **출처와 무관하게 라이선스가 명시적으로 열린 이미지**여야 함. 진짜 병목.
2. **네이버·해외 블로그(The Artful Parent 등)는 "저작권 클린 소스"가 아니라 "발견·영감 채널".** 네이버=포털(원저작권은 원매체), The Artful Parent=미국 상업 블로그(국내한정 정책 충돌). 영감만 받고 실제 인용은 국내 1차 출처로 치환.

### 이미지용 라이선스 클린 풀 (별도 메모, 본 문서 범위 밖이지만 연결)
- 국내: 공공누리(KOGL, kogl.or.kr — 제1유형=출처표시만 하면 상업·변형 무료), 공유마당(gongu.copyright.or.kr), 문화포털/e뮤지엄
- 글로벌 CC0 스톡: Unsplash·Pexels·Pixabay·Openverse
- 권장: 기사별 매칭은 솔로 과부하 → **카테고리/콘텐츠타입 단위 대표 이미지 큐레이션 + 폴백 비주얼(A)** 하이브리드

## Part 3. V2 인플루언서 화이트리스트화 초안

### 핵심 통찰: "인플루언서"는 두 종류 — 자격증 있으면 인플루언서가 아님
- 하정훈(삐뽀삐뽀119, 서울대 소아청소년과·백신위원장·26만 구독) = **그냥 Category B에 추가**하면 끝. 신규 워크플로 불필요.
- 진짜 V2 회색지대 = **자격증 없지만 가치 있는 크리에이터**: 베싸TV(근거 기반), 차이의놀이·대디를위한가이드(놀이·활동 소재).

### 3티어 프레임워크
| 티어 | 정의 | 예 | 신뢰도 | 허용 범위 |
|---|---|---|---|---|
| **T0 자동자격** | 자격증 전문가 채널 | 하정훈·오은영(기존 B) | 0.80~0.90 | B로 흡수, 심사 불필요 |
| **T1 근거기반 크리에이터** | 자격증 X, 출처 인용·연구 기반 | 베싸TV (babyscience.io) | 0.70~0.78 | "정보·발달" 콘텐츠 *조건부* |
| **T2 경험·활동 크리에이터** | 놀이·활동 아이디어·경험담 | 차이의놀이·대디를위한가이드 | 0.65~0.72 | **소재만**, 상업 분리 엄격 |

(기존 A 0.85~0.95 / B 0.80~0.90 / C 0.75~0.85 **아래**로 배치 — 신뢰 위계 유지)

### 채널 심사 체크리스트 (1회 onboarding)
- ☐ 협찬 투명성 (광고·협찬 표시 안 하면 탈락)
- ☐ 근거 제시 (출처·연구, T1 필수)
- ☐ 자극성 없음 (공포·과시·"안 하면 뒤처짐" 금지 — 톤 가드레일)
- ☐ 검증가능성 (의학·발달 주장이 1차 출처와 대조 가능)
- ☐ 중립성 (정치·종교·특정 상품 편향 없음)

### 콘텐츠 단위 협찬 게이트 (매 발행)
- 협찬·광고 명시 콘텐츠 = 자동 배제 (기존 정책 강화 적용)
- 배선: source-level 티어 + per-item 협찬 플래그 2단 (파이프라인 source 신뢰도 + QA 단계 재활용)

### 격리 원칙 (기존 정책 정합)
- 인플루언서 콘텐츠는 "정보·소재"로만. "이 학원/제품 추천"은 편집 콘텐츠로 절대 X.
- 추천·매칭은 제휴 광고 레이어로 격리 (톤 가드레일 §, STRATEGY 제휴 BM).
- = §10.1 "검증된 책임" 해자 보존 (AI 노동, 사람이 채널 심사 책임).
