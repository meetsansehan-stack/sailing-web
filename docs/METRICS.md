# 성공지표 (단일 소스)

> 베타 = **학습 프로브**(STRATEGY §8 시퀀싱). 측정 목적 = "진통제 *공명* 신호인가, 비타민인가"를 가른다.
> 정량(익명 이벤트)은 closed 베타(10~30명)에선 *방향*만 — **정성(인터뷰 코딩, ICP_INTERVIEW §8)이 주력.**
> 도구: 자체 익명 적재(day1) → **운영 대시보드 `/dashboard?key=`** (`GET /api/analytics/summary`).

## 0. 북극성

- **앱 본체 북극성** = "내 아이 청사진" (STRATEGY §1, 별개 트랙).
- **웹(Sailing Web) 베타 북극성** = **주간 되돌아오는 부모** (시점 기반 relevance=키데이트 캘린더가 엔진). 공명 = 재방문·완독·"놓친 거 없나" 충족.

## 1. AARRR 컷 (MVP = Activation + Retention, STRATEGY §9·§10)

| 단계 | 베타에서 보는 것 | 비고 |
|---|---|---|
| Acquisition | 초대 N·유입 경로 | **유통과 한 묶음**(안 그러면 트래픽↓ 원인 불명) |
| Activation | 첫 방문 캐치업 완료(다중 기사 열람·CTA 노출) | |
| **Retention** ★ | **주간 재방문**, "미리 준비" 재방문·알림 옵트인 | 핵심 = PMF 테스트 자체 |
| Referral | 실제 전달(소개) | 오가닉0의 진짜 검증. 후순위지만 측정 |
| Revenue | — | **출시 전 금지**(STRATEGY §10) |

## 2. 공명 지표 ↔ 이벤트 매핑 (대시보드에서 조회)

| 지표 | 이벤트 | 의미 | 상태 |
|---|---|---|---|
| 트래픽·고유 방문자 | `page_view` (+`anonId` distinct) | 도달 | ✅ 적재·대시보드 |
| 읽기(기사 열람) | `page_view` path=`/articles/*` | 캐치업·둘러보기 깊이 | ✅ (대시보드 인기경로) |
| **원문 클릭(intent·신뢰)** | `outbound_click` | "출처로 가서 읽는다"=깊은 신뢰 신호 | ✅ 이 작업서 배선 |
| 구독 퍼널 | `cta_impression`→`cta_click`→`subscribe_success` | 오디언스 전환 | ✅ 적재·대시보드 |
| 체류시간·스크롤 깊이 | (미적재) | 몰입 vs 가둠(STRATEGY §11.1) | 🔲 후속(세션·scroll 적재) |

> `article_open`은 타입엔 있으나 별도 발화 안 함 — `page_view`+경로로 이미 잡혀 **중복 배선 안 함**(이중집계 방지).

## 3. PMF 테스트 (STRATEGY §8)

1. **Sean Ellis 40%** — "내일 사라지면 매우 아쉽다" ≥ 40% (인터뷰 §6·§8 + 후속 in-app micro-survey).
2. **전달 행동** — 실제 소개 ≥ 인터뷰의 30% (오가닉0의 진짜 검증, "좋아요"가 아니라 *행동*).
3. **리텐션 곡선 평탄화** — 주차가 지나도 되돌아오는 비율이 0으로 수렴하지 않음.

## 4. 베타 kill/learn 기준 (★ 베타 *전에* 고정 — STRATEGY 가드레일 3)

- **전제 가드레일**: ① 진통제("미리 준비"·마감 레이더)가 1급 포함 ② 유통과 한 묶음 ③ 기준 사전 고정.
- **kill/피벗 신호**(복합): 리텐션 0 수렴 **+** Sean Ellis <40% **+** 소개 0 **+** "미리 준비" 재방문·옵트인 0.
- **persevere 신호**: 위 중 2+가 양(+), 특히 진통제(키데이트) 경로에서 재방문.
- ⚠️ **"개선(퍼널 최적화)"이 "필요 검증(need·WTP)"을 덮지 않게** — need·WTP는 라이브여도 *물어봐야* 나옴(인터뷰·micro-survey). growth 배경의 최대 유혹 = 병목이 funnel이라 착각(병목=공명+유통).

## 5. 베타 현실 목표 (closed 10~30명)

작은 N → 정량은 *방향*, 정성이 판정. 잠정 타깃(절대선 아님, 분해용 기준선):

- 주간 재방문 ≥ 30% (초대자 중)
- 다중 기사 열람 세션 ≥ 40% (캐치업이 실제로 일어남)
- 원문 클릭률(`outbound_click`/방문) ≥ 10% (신뢰·intent)
- Sean Ellis ≥ 40% (인터뷰), 소개 ≥ 30%

## 6. 측정 도구 스택

- **정량**: 자체 익명 이벤트(day1) → `/dashboard?key=<ADMIN_API_TOKEN>` (구독 퍼널·일별 추이·인기 경로·고유 방문자). PII 0.
- **정성**: ICP 인터뷰 + **코딩 시트 = `docs/ICP_INTERVIEW.md` §8**(The Mom Test — 과거 행동·돈·감정·verbatim, 칭찬 제외). 판정 기준 동 문서 §8.
- **need·WTP**: in-app micro-survey (후속) — 라이브여도 물어봐야 나옴.
