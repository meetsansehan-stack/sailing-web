## 1. 한 줄 정체성
아이의 성장이라는 **항해**를, 부모가 자신감 있게 나아가도록 — **오늘의 좌표부터 먼 수평선까지** 함께하는 브랜드.

| 항목 | 내용 |

## 3. 슬로건의 뜻 (제품 구조 은유)
"오늘의 좌표"(데일리 큐레이션) → "먼 수평선"(장기 성장·청사진). 매일의 캐치업과 먼 여정의 지도를 한 줄에 담는다.

## 4. 브랜드 아키텍처
```
Sailing  ──  총괄 브랜드 (웹 + 앱)
│
├─ Sailing Web (이 repo · 가제 "페런트웹")
│    ├─ 데일리 카드 큐레이션  →  매일의 "좌표"
│    └─ '아이의 항해' 롱폼 섹션  →  여정 "에세이" (maily 뉴스레터가 이 섹션으로 편입)
│
├─ Sailing App = '내 아이 청사진'  →  항해 "해도" (교육·여정 플래너)
│
└─ (미래 서브 브랜드)
```
- **'아이의 항해'**(기존 뉴스레터)는 웹의 롱폼 섹션으로 흡수된다. 그 섹션의 하위 슬로건 = "Physicality over Pixels"(AI 시대 몸·감각). **우산 슬로건이 아니라 한 섹션의 색.**
- 포맷 라우팅: 데일리·속보(Policy·Event) = **카드** / 발달·관계·사유형('실패의 힘' 등) = **롱폼**.

## 5. 항해 어휘 체계 (UI·카피 공용)
좌표 · 나침반 · 해도 · 수평선 · 등대 · 파도 · 닻. (뉴스레터 섹션명이 이미 파도·나침반·닻·수평선을 사용 — 일관 유지.)

## 6. 계승하는 브랜드 DNA (포맷 무관)
'아이의 항해' 편집실 지침(검증된 자산)에서 우산으로 끌어올린 원칙. 상세·집필 적용은 [brand-voice.md](brand-voice.md).
- **죄책감 유발 금지** + 면죄부 한 줄 (양육 콘텐츠 최강 차별점)
- **불안 공감✅ vs 공포 자극❌** (대상이 '육아 일상'=공감 / '미래·위협'=공포 배제)
- 능동태(주어=사람), 영어 번역투 지양
- 대상 나이를 흐름 안에 자연스럽게(별도 연령표 X)
- **"부모의 나침반" so-what·WOW 테스트** = 편집 품질 바
- 현재 인물·이슈 오프닝 + 고전 연구 뒷받침
- 팩트체크 엄격성(1차 출처, 추정 금지)
- 차분한 이해·발달 중심, "둘 다 품되 선 긋기"([../CLAUDE.md](../CLAUDE.md) 톤 가드레일과 일치)
