# 6~7월 출시 콘텐츠 플랜 — 공공 여름 (Sailing 적합)

> 수요 발굴(공공 포커스, [DEMAND_PUBLIC.md](DEMAND_PUBLIC.md)) 기반. 상업시설 제외, 국공립·지자체·도서관만.
> **별도 메뉴/특집 아님** — 각 시리즈는 *하나의 큐레이션 기사*(Guide/Insight)로 기존 카테고리에 편입.
> 데이터는 [public-summer-seed.ts](../packages/db/scripts/demand/public-summer-seed.ts)에 축적 → 큐레이션·집필 파이프라인 입력.

## 발행 원칙
- 7~8월 피크라 **6월 말~7월 초 발행 = 미리 준비**(올라타기). 신청·예매형은 더 선행.
- 출처는 화이트리스트(공식)만, 일정은 집필 전 WebFetch 재검증(`dateConfirmed=false` 큐).
- 프레이밍: "흩어진 공공 일정을 한곳에" = 우리 차별점. "아이랑/어린이/무료" 명시.

## 시리즈 4개 (각 = 1 기사, 기존 카테고리 편입)

### 1. 🗺️ 수도권 무료·공공 물놀이장 지도
- category `play` / contentType Guide(또는 Insight) / 발행 6월 말
- 내용: 한강 물놀이장 4곳(잠실·광나루·양화·난지, 6세↓ 무료) + 구·시 무료 물놀이장·바닥분수·한강 어린이 수영장 통합. 지역·요금·개장일.
- 근거: 물놀이터 skew 1.32·여름집중 91% / 한강수영장 월 1.6만·여름집중 92%. 구·시마다 흩어짐 = 통합가치 최고.

### 2. 📚 도서관·문화센터 여름방학 프로그램 총정리 (수도권)
- category `books`/`others` / contentType Guide / 발행 6월 말~7월 초(신청 선행)
- 내용: 국립어린이청소년도서관 + 시립·구립 도서관 + 지자체 문화센터 여름 독서·체험 프로그램 신청 안내. D-day 알림 대상.
- 근거: 문화센터 skew 1.41·도서관 여름방학 1.58(여름집중 100%).

### 3. 🔬 과학관·박물관 여름 특별전·특별교육 모음
- category `play`(체험)/`shows`(전시) / contentType Event·Insight / 발행 7월
- 내용: 국립과천과학관 여름방학 특별교육 + 어린이박물관·국립과학관 여름 특별전. (특별전 일부 9월까지 — 시점성 김)
- 근거: 국립과천과학관 월 3.8만·경쟁 낮음(SEO 기회)·8월 피크 / 어린이박물관 17k·7월 피크.

### 4. 🎭 여름방학 어린이 공연·전시 캘린더 (공공 공연장)
- category `shows` / contentType Event / 발행 6월 말(예매 선행)
- 내용: 시·구 문화재단·공공 공연장 어린이 뮤지컬·공연·인형극. (사설 제외)
- 근거: 어린이뮤지컬 월 3만·7월 피크(어린이날 아닌 여름이 진짜 피크).
