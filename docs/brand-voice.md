## 0. 포맷 두 갈래 (먼저 읽기)
같은 보이스, 다른 그릇. [BRAND.md §4](BRAND.md) 포맷 라우팅과 짝.
- **카드뉴스** (데일리 큐레이션, 기본): 불릿·**친근한 존대**·한눈 스캔. **따뜻함은 문장 길이가 아니라 *단어 선택*과 *양육자 시사점 한 줄*에서 나온다.** (2026-05-29 CPO 확정: 기존 "명사형 종결"이 너무 딱딱 → **친근한 존대**로 전환. "무료니까 꼭 확인하세요!" 식의 따뜻한 안내 톤. 단 *혜택·사실 프레이밍*이지 공포·재촉 아님.)
- **롱폼** ('아이의 항해' 섹션·사유형 주제): 편집실 지침의 5섹션 내러티브 계승(AI·몸 색은 일반화). 분량 4,000~5,500자.

아래 §1~§3은 두 포맷 공통. §4가 포맷·타입별 변주.

## 1. 브랜드 인격 (persona)
- **한 문장**: "아이가 선장, 부모가 조타수인 항해에서 — 길을 객관적으로 가리키는 나침반이자, 멀리 내다보는 북극성."
- **역할 위계**:
  - **아이 = 선장** — 항해의 주인. 어디로 갈지는 아이의 삶.
  - **부모 = 조타수** — 키를 잡고 가장 적극적으로 돕는 역할.
  - **Sailing = 중립적 항법 장치·지표** — 나침반·북극성·해도. 조타수가 길을 객관적으로 보고 멀리 내다보게 돕는다. **길을 대신 정하지 않는다.** → "중립 데이터 매체" 포지셔닝과 일치.
- **따뜻함(동행) + 객관(항법) 공존**: *매너는 따뜻하게, 내용은 객관적으로.*
- **우리가 아닌 것 (NOT)**:
  - **"먼저 가본 선배의 경험담" X** — 경험·권위가 아니라 객관적 도구·중립 데이터
  - 훈계조 X ("~해야 합니다" 식 지시)
  - 불안·공포 조장 X ("지금 안 하면 늦는다")
  - 전문가 권위주의 X (어려운 용어로 거리 두기)
  - **죄책감 유발 X** (과거형 "못 해줬다")
  - 트렌드 추종·과시 X

## 2. 톤 & 매너
- **친근한 존대 (기본)**: 카드 불릿도 **"~해요/~하세요/~좋아요"** 친근한 존대로 끝낸다. 마지막 불릿 또는 "양육자 시사점" 한 줄에서 **동행의 온기**(안심·죄책감 면죄부)를 준다. 정보 나열로 끝내지 않는다. (예: "신청해야 받을 수 있어요 — 꼭 챙기세요!")
- **호칭**: 양육자를 무겁게 직접 호명하지 않는다("부모님께서는~" X). 주어 없이 자연스러운 존대("~해 주세요"), 롱폼은 담백한 2인칭 최소 사용.
- **능동태**: 주어를 사람(아이·양육자)으로. "감각이 쌓여요" 같은 무생물 수동 지양.
- **영어 번역투 지양**: "~하는 것이 필요합니다" 같은 번역투·딱딱한 격식체 대신 **자연스러운 구어 존대**("~하면 편해요", "~만 확인해 주세요").
- **이모지·구어체**: 절제(담백·우아하되 따뜻하게). 본문 이모지 절제하되 안심 톤 마무리에 한 개 정도 (필요시 사용 🙂). 항해 어휘(좌표·나침반 등)는 섹션·메타에서만.

## 3. 신뢰 신호 (trust)
- **출처 제시**: 출처(`source`)를 본문 맥락에 자연스럽게("교육부 발표 기준~"). 단정적 수치엔 출처를 붙인다.
- **단정 vs 유보**: 확정 정책·발표 = 단정("3월 시행"). 추정·연구 = 유보("연구마다 다르지만", "일반적으로").
- **불안 없이 중요도 전달**: 중요도는 *사실·규모·시점*으로 ("전국 확대", "신청 마감 3월"). 감정 증폭어("충격·비상·큰일") 금지. → CLAUDE.md "불안 공감✅ vs 공포 자극❌"과 동일.
- **죄책감 면죄부**: 양육자가 위축될 지점엔 "오늘부터" 미래지향 한 줄. ("이미 늦음" 프레이밍 금지.)

## 4. 포맷·타입별 미세조정
- **Policy** (차분·정확): 카드. "달라지는 점/양육자 체크"에서 우리 집 적용 한 줄.
- **Guide** (친근·실용): 카드. 단계·체크리스트, "오늘부터" 톤.
- **Event** (설렘·정보): 카드 + 일시·장소·예약 메타.
- **Insight·Market** (담백·중립): 카드. **"둘 다 품되 선 긋기"** — 사교육·또래비교는 순위 아닌 *참고 분포*, 통계 비주얼화. 공포·과시 금지.
- **사유형/발달·관계** ('실패의 힘' 등): 롱폼. **"부모의 나침반"** so-what·WOW 테스트 적용(판단기준·시기·행동전환형, 일반 조언 금지). 현재 인물·이슈 오프닝 + 고전 연구 뒷받침.

## 5. 좋은 예 / 나쁜 예 (카드) — 친근한 존대 기준
**Policy — 늘봄학교 확대**
- ✅ `## 양육자 체크` / `- 신청은 보통 3월 2주 온라인이에요 — 일정만 미리 확인해 주세요` / `- 맞벌이라면 하원 공백을 메우는 데 유효해요`
- ❌ `- 지금 신청 안 하면 자리 없음, 서두를 것` (공포·재촉) / `- 신청 3월 2주 온라인, 일정 확인 필요` (명사형 — 너무 딱딱)

**Insight — 영어유치원 참여율**
- ✅ `- 수도권 5세는 약 N% 정도가 다녀요 (빠름/늦음에 정답은 없어요)`
- ❌ `- 안 보내면 또래보다 뒤처짐` (순위·불안)

**죄책감 면죄부**
- ✅ `- 지금 시작해도 충분해요 — 오늘 저녁부터 가능하답니다 🙂`
- ❌ `- 더 일찍 챙겼어야 하는 시기` (과거형 죄책감)
