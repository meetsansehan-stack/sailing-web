# 메일 리포맷 실험안 — 비타민 → 진통제

> **목적**: STRATEGY §8.4-①. 기존 "아이의 항해"(maily, 주1회, 구독 16명)를 **재구축 없이 포맷만** 진통제로 바꿔, "안 와닿음"이 "챙겨야겠다"로 바뀌는지 **행동 델타**로 검증.
> **0원·코드 0** — 기존 채널·도구 그대로. [[ICP_INTERVIEW]]와 한 묶음(자극물 = 이 메일).

---

## 1. 가설 & 측정 철학

- **H1**: 진통제 포맷("이번 주 놓치면 안 될 신청·마감 3 + 입학 준비 1")은 비타민 큐레이션보다 **능동 반응**(답장·포워드·질문·요청)을 높인다.
- **측정 = 행동 우선**(§8.1): "좋아요/유익해요"는 **집계 제외**(politeness 편향). 카운트하는 건 **답장·포워드·공유·구체 요청** + Sean Ellis.
- ⚠️ **n=16, 대부분 지인 → 통계 아님, 방향 신호**. 그래서 ICP 인터뷰(지인 밖)와 병행해야 결론. 가능하면 취학 전후 세그 신규 구독 5~10명 유입(맘카페/오픈채팅) 후 그들 반응을 별도 태깅.

## 2. 설계 (within-subject, 2~3주)

| 주차 | 포맷 | 목적 |
|---|---|---|
| W0 (직전 1~2회) | 기존 큐레이션 | 베이스라인(현 반응률) |
| W1~W3 | **진통제 포맷** | 델타 측정 |

- 같은 리스트에 연속 발송 → 본인 전후 비교. 발송 요일·시간 고정(변수 통제).
- 매회 **명시적 행동 유발 1개**씩 심기(아래 §4).

## 3. 진통제 포맷 스펙

**제목(subject)**: D-day·구체 마감 노출. 공포 아님·혜택 프레이밍.
- 좋은 예: `[이번 주] 늘봄 신청 마감 D-3, 놓치면 한 학기 기다려요`
- 나쁜 예: `이번 주 육아 소식 모음` (비타민·뭉뚱)

**본문 구조** = 마감 3 + 준비 1. 각 항목 6요소:
1. **무엇** (한 줄, 친근한 존대)
2. **누가 해당** (예: "○○년생 / 맞벌이 / 첫 초등")
3. **언제까지** (D-day, 굵게)
4. **어디서** (원문·신청 링크 — 필수)
5. **왜 중요** (한 줄: 놓치면 어떻게 되는지 — 사실·규모·시점, 공포 X)
6. **자동 vs 직접** ("자동으로 오는 것 ❌ / 직접 신청해야 받는 것 ✅" — §8.3 핵심 메커니즘)

**톤 가드(brand-voice)**: 친근한 존대("꼭 확인하세요!") · 죄책감 면죄부("지금부터 챙기면 충분해요", "이미 늦음" 금지) · 불안 공감 O / 공포 자극 X · 선 긋기(사교육=통계, 학원 나열 X).

## 4. 행동 유발 장치 (매회 1개)

- **W1**: 답장 유도 — "이 중에 **이미 아는 거/처음 듣는 거** 답장으로 알려주실래요?" (engagement→대화)
- **W2**: 포워드 유도 — "주변에 **취학 앞둔 부모** 있으면 이 메일 그대로 전달해 주세요" (referral 행동 = 오가닉0 깨는 신호)
- **W3**: 요구 발굴 — "다음에 **어떤 마감을 미리 알려드릴까요?** 한 줄 답장 부탁해요" (need pull 측정)

## 5. 지표 & 집계

| 지표 | 종류 | 비고 |
|---|---|---|
| **답장 수/율** | 행동(primary) | 내용도 코딩(질문>감상) |
| **포워드/공유** | 행동(primary) | W2 핵심 |
| **구체 요구** | 행동(primary) | W3, "X 알려줘" |
| 오픈율 | engagement(2차) | need 아님 |
| 링크 클릭율 | intent(2차) | 어떤 항목이 끌리나 |
| Sean Ellis(분기 1회 말미) | PMF | "사라지면 아쉬움 매우/조금/별로" |
| 구독 변화(+/-) | 생존 | 신규 세그 유입분 분리 태깅 |

## 6. 판정 (kill/learn 미리 박기)

- ✅ **유지·강화**: 진통제 주차 **능동 반응(답장+포워드+요구) ≥ 베이스라인 2배** 또는 포워드 ≥1건 발생(지인 리스트에서 자발 전달은 강한 신호).
- ⚠️ **재조준**: 반응은 늘었는데 A(준비)/B(마감) 한쪽만 끌리면 → 끌리는 쪽으로 좁힘.
- ❌ **가설 약화**: 포맷 바꿔도 능동 반응 거의 0 → 진통제 가설 자체 or 유통(이 리스트=지인 웅덩이) 문제. ICP 인터뷰 결과와 교차.
- ⚠️ 어떤 경우든 **"좋다"는 카운트 안 함**.
