# Toss Feed 분석 & 세일링 뉴스 적용 패턴

> 레퍼런스: https://toss.im/tossfeed · 분석일 2026-05-29
> 용도: 세일링 뉴스(웹) 레이아웃·큐레이션·콘텐츠 노출 설계 참고

---

## 1. Toss Feed 구조 요약

| 영역 | 내용 |
|---|---|
| **내비게이션** | 에디토리얼 탭 4개 (토스피드 에디션 / 금융의 모든 것 / 토스의 모든 것 / 계산기) — *적은 수, 큰 묶음* |
| **Hero = 고정 픽** | 풀너비 대표 이미지 + 텍스트 오버레이 + **"이 주의 콘텐츠"** 라벨 (에디터 선정 1개를 크게) |
| **지금 많이 보는 콘텐츠** | 순위 번호 + 제목 + "새로 나온" 태그 + 구분선 = **미니멀 랭크 리스트** (카드 아님) |
| **큐레이션 섹션 다발** | 인기("사람들이 주목하는") / 주제별("부동산 투자에 관심 있다면") / **필진 큐레이션** / **오리지널 시리즈** / 최신("방금 올라온") |
| **디자인 톤** | 넉넉한 여백 · 굵은 산세리프 제목 · 흑백 기조 + 파랑 포인트 · **섹션마다 카드 형태 상이**(큰 이미지 / 텍스트만 / 썸네일+제목) |

---

## 2. 세일링 뉴스에 가져올 패턴 5

1. **"이 주의 픽" Hero 1개** (큰 이미지 + 오버레이) — 기존 `hot-issue.ts`(`selectHotArticle`)로 바로 구동 가능 (🔥 핵심 이슈를 featured로)
2. **섹션마다 노출 형태 차별화** — 현재 홈은 *같은 카드 그리드 반복*이라 단조로움. 피처 큰 카드 / 랭크 텍스트 리스트 / 썸네일 리스트로 다양화 = "큐레이션 매체" 인상
3. **주제·시즌 큐레이션 섹션** ("취학 앞두고 있다면" 등) → 미리 준비와 연결
4. **필진/출처 큐레이션 + 오리지널 시리즈** → 출처 풀(오은영·EBS·기관) + '아이의 항해' 롱폼
5. **랭크 리스트("많이 보는")** → viewCount 필요 = **V2** (지금은 hot-issue 신호로 대체)

## 3. 패턴 3 상세 — 주제·시즌 큐레이션

**기준 = 상황·시기.** 카테고리/날짜를 가로질러, *지금 이 부모의 처지*에 맞는 묶음을 큐레이션.

### 예시 ① 시즌(시기) 기반 — 미리 준비와 직결
```
🎒 지금, 취학을 앞두고 있다면              [미리 준비 전체 →]
 · [정책] 2027 초등 취학통지·예비소집 일정
 · [가이드] 늘봄·돌봄 신청, 맞벌이 1순위
 · [인사이트] 입학 전 한글, 꼭 떼야 할까 (참고 분포)
```
→ 이 섹션 자체가 미리 준비(/radar)로 가는 입구.

### 예시 ② 상황(잡) 기반
```
👫 맞벌이라면 꼭 챙길 것
 · 긴급돌봄·아이돌봄서비스 신청법
 · 방학 중 돌봄 공백 메우기
```

### 예시 ③ 계절 기반
- "여름방학, 어떻게 보낼까" (체험·캠프·돌봄)
- "환절기 우리 아이 건강"

> **구현**: `category` + `contentType` + `tags` + 시즌 윈도우로 규칙 기반 묶거나, 운영자가 수동 큐레이션. 현재 데이터로 바로 가능.

## 4. 패턴 4 상세 — 필진/출처 큐레이션 + 오리지널 시리즈

**기준 = 누가 말하나 / 어떤 시리즈인가.** 사람 필진 대신 **검증된 출처 풀**과 **자체 시리즈** 활용.

### 예시 ④ 출처·전문가 큐레이션 — "신뢰"를 둘러볼 수 있게
```
👩‍⚕️ 전문가가 짚어주는 발달               [전문가 글 더 →]
 · 김붕년 "초등 입학 전 불안, 이렇게"
 · 조선미 "떼쓰는 아이 대하는 법"
 · 오은영 칼럼 ...
```
```
🏛️ 공영방송·공공기관 표준 정보   (EBS·복지부·교육부)
📖 그림책 평론가의 선택        (한미화·김지영·어린이도서연구회)
```
→ 카드마다 작게 붙던 **출처 배지·credibilityScore를 하나의 "둘러보는 큐레이션"으로 격상.** 맘카페와 가장 차별화되는 지점(신뢰)을 *눈에 보이게*.

### 예시 ⑤ 오리지널 시리즈 — 따라 읽게 만드는 자체 콘텐츠
```
📖 오리지널 · 아이의 항해                  [시리즈 →]
 · (롱폼) "실패의 힘 — 좌절을 다루는 법"
```
- **아이의 항해**(사유형 롱폼, 브랜드 자산) = 오리지널 시리즈 1호
- 추가 후보: "이번 주 정책 브리핑"(주간 정리), "30초 발달 노트"(연령별)
- 시리즈는 *팔로우*하게 만들어 재방문(retention) 유도

> **구현**: `source`(출처 풀 소속)로 묶으면 ④ 바로 가능. ⑤ 시리즈는 `tags`나 시리즈 필드로. (사람 저자 필드는 없지만 `source`가 그 역할.)

---

## 5. 한 줄 정리

- **3번 = "당신 상황에 맞는 묶음"** (시즌·잡 → 미리 준비와 연결)
- **4번 = "믿을 만한 출처·우리 시리즈 묶음"** (신뢰 브랜드 + 아이의 항해 → 재방문)
- 둘 다 홈을 *날짜 카드 나열* → **"큐레이션된 매체"**로 바꾸는 레버.
